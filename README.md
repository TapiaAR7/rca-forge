# RCA Forge

Free Root Cause Analysis (RCA) builder for industrial teams.

RCA Forge helps engineers, managers, maintenance teams, quality teams, automation teams, and interns build structured RCA reports using:

* Problem statements
* Fishbone diagrams
* 5 Whys
* Evidence review
* Root cause validation
* Corrective and preventive actions
* Export-ready reports

## Purpose

Many RCA reports jump too quickly from a problem to a conclusion.

RCA Forge is designed to guide the user through a cleaner process:

**Problem → Fishbone → Evidence → 5 Whys → Root Cause → Actions → Export**

The goal is not to guess the root cause automatically.
The goal is to help users organize the investigation, challenge assumptions, document evidence, and create a professional RCA report.
